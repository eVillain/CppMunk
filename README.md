CppMunk: A C++11 wrapper for Chipmunk2D Physics (https://chipmunk-physics.net/).  

    --- Current feature set ---  
Most features from Chipmunk2D 7.0.1 are fully wrapped into C++.  
Collision callbacks are also supported using the Arbiter class.
